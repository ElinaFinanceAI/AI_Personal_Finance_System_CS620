# Personal Finance Management System

This project is a partial implementation of a Personal Finance Management System developed for CS 620.

## Modules
- expense_tracker.py
- budget_manager.py
- alerts.py
- test_system.py

## Description
The system allows users to:
- add expenses
- calculate total spending
- compare spending to a budget
- receive alert messages
